# **App Name**: SATS Dashboard

## Core Features:

- Volume Display: Display transaction volume for today, yesterday, last 7 days and last 30 days.
- Payment Summary: Show total balance and payment summary graph.
- Transactions List: List all transactions with details.
- User Authentication: Implement user authentication for secure access to the dashboard.
- Data Fetching: Fetch transaction data from an external API.

## Style Guidelines:

- Primary color: #4B6BFB, a vibrant blue to convey trustworthiness and innovation.
- Background color: #F5F7FB, a light, desaturated shade of blue that is easy on the eyes.
- Accent color: #FF7A59, a warm orange to draw attention to key metrics and actions.
- Headline Font: 'Poppins', a geometric sans-serif, to be used for titles, headings and data.
- Body Font: 'Inter', a grotesque sans-serif, to be used in descriptions and additional text if needed.
- Use a left sidebar for navigation and the main area for displaying dashboard elements.
- Use cards with rounded corners for the volume display.
- Use simple, modern icons for navigation elements.
